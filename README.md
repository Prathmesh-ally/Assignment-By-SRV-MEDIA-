# Premier Schools Exhibition - Landing Page
# Note:- All the photos folder should come under assets foleder as shown in project structure 
A modern, responsive landing page designed for the "Premier Schools Exhibition" in Gurugram. This project serves as an event portal where parents can view participating schools, register for the event, and pre-schedule appointments.

* **Responsive Design:** Fully optimized for desktops, tablets, and mobile devices.
* **Modern UI/UX:** Features a vibrant gradient background and clean typography using the 'Montserrat' font family.
* **Animations:**
    * **Infinite Marquee:** Vertical scrolling animation for event gallery images.
    * **Hover Effects:** Smooth transitions on buttons, cards, and school categories.
* **Glassmorphism:** The enquiry form features a modern "frosted glass" effect.
* **Sections Included:**
    * Hero Section with Event Details (Date & Venue).
    * Live Scrolling Image Gallery.
    * Registration/Enquiry Form.
    * Key Statistics (Legacy, Footfall, etc.).
    * Participating Schools (Horizontal Scroll).
    * School Categories (Pre-school to International).
    * Pre-schedule Appointment Call-to-Action.
    * Parent Testimonials.
    * Footer with Contact Info & Social Links.
